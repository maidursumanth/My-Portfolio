## ⚙️ Installation<br>

### Clone Repository<br>

```bash<br>
git clone https://github.com/yourusername/portfolio-website.git
```

### Navigate to Project Folder<br>

```bash<br>
cd portfolio-website<br>
```

### Install Dependencies<br>

```bash<br>
npm install<br>
```

### Start Development Server<br>

```bash
npm run dev<br>
```
<br>
### Build For Production<br>

```bash<br>
npm run build<br>
```
